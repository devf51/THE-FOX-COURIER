# 🦊 THE FOX COURIER — คู่มือเกม

> **ประสบการณ์ส่งของสุดสยอง**
> เกมสยองขวัญแนว Top-down ที่คุณต้องส่งพัสดุให้ครบทุกบ้าน ท่ามกลางความมืดและสิ่งลึกลับที่ซุ่มอยู่ในเงามืด

---

## 📖 เนื้อเรื่อง

คุณคือ **พนักงานส่งของกะดึก** ที่ต้องทำงานส่งพัสดุในหมู่บ้านลึกลับ ทุกคืนจะมืดมิดและเต็มไปด้วยอันตราย — **ผีและสิ่งเหนือธรรมชาติ** คอยหลอกหลอนอยู่ตามทาง คุณต้องส่งพัสดุให้ครบทุกบ้านก่อนเวลาจะหมด โดยไม่ให้ถูกจับได้!

---

## 🎮 การควบคุม

| ปุ่ม | การทำงาน |
|------|----------|
| `W` | เดินขึ้น |
| `A` | เดินซ้าย |
| `S` | เดินลง |
| `D` | เดินขวา |
| `SHIFT` (ค้าง) | วิ่ง (ใช้พลังงาน) |
| `F` | โต้ตอบ / เคาะประตู |
| `E` | ตรวจสอบกระเป๋า |
| `ESC` | หยุดเกมชั่วคราว |

---

## 🌙 ระบบคืน (Night System)

เกมแบ่งเป็น **3 คืน** โดยแต่ละคืนจะยากขึ้นเรื่อยๆ:

| คืน | จำนวนพัสดุ | ความเร็วศัตรู | ความยาก |
|-----|-----------|--------------|---------|
| คืนที่ 1 | 3 ชิ้น | ช้า | ★☆☆ |
| คืนที่ 2 | 4 ชิ้น | ปานกลาง | ★★☆ |
| คืนที่ 3 | 5 ชิ้น | เร็ว | ★★★ |

- เล่นผ่านแต่ละคืนจะ **ปลดล็อคคืนถัดไป** โดยอัตโนมัติ
- ความคืบหน้าจะ **บันทึกอัตโนมัติ** — ปิดเกมแล้วกลับมาเล่นต่อได้
- เล่นจบคืนที่ 3 จะได้หน้าจอ **ยินดีด้วย!** พิเศษ 🏆

---

## 💚 ระบบพลังงาน (Stamina)

ใต้เท้าตัวละครจะมี **แถบพลังงาน** แสดงอยู่ตลอดเวลา:

- 🟢 **เขียว** (100%-50%) — พลังงานดี วิ่งได้เต็มที่
- 🟡 **เหลือง** (50%-25%) — เริ่มหมด ควรหยุดพักให้ฟื้นตัว
- 🔴 **แดง** (ต่ำกว่า 25%) — อันตราย! เกือบหมดแล้ว

### กลไกพลังงาน:
- **วิ่ง** (กด SHIFT ค้าง) จะ **ลดพลังงาน** ลงเรื่อยๆ
- **เดินปกติ** จะ **ฟื้นฟูพลังงาน** อย่างช้าๆ
- **หยุดนิ่ง** จะ **ฟื้นฟูพลังงาน** เร็วขึ้น
- ถ้าพลังงาน **หมด** → ต้องรอฟื้นตัวก่อนจึงจะวิ่งได้อีกครั้ง

> ⚠️ **เทคนิค**: อย่าวิ่งจนพลังงานหมด! เก็บไว้สำหรับหนีผี

---

## 🏠 การส่งพัสดุ

1. ดู **ลูกศรชี้ทิศทาง** ที่ด้านบนหน้าจอ — จะชี้ไปยังบ้านที่ใกล้ที่สุดที่ยังไม่ส่ง
2. ดู **Minimap** ที่มุมจอ — จุดสีแสดงตำแหน่งบ้านแต่ละหลัง
3. เดินเข้าใกล้บ้าน → จะมีข้อความ **"กด F เพื่อเคาะประตู"** ขึ้นมา
4. กด `F` → แถบ **ความคืบหน้าการเคาะ** จะปรากฏ (ใช้เวลา 2 วินาที)
5. เคาะเสร็จ → ระบบจะ **ส่งของอัตโนมัติ** และนับจำนวนที่ส่งเพิ่ม
6. ส่งครบทุกบ้าน → **ผ่านด่าน!** 🎉

### สถานะบ้านบน Minimap:
- 🔴 **แดง** — ยังไม่ได้ส่ง
- 🟢 **เขียว** — ส่งแล้ว

## 👻 ศัตรู

ศัตรูจะเดินลาดตระเวนตามแผนที่ มีพฤติกรรมดังนี้:

| สถานะ | พฤติกรรม |
|--------|----------|
| 🔵 ลาดตระเวน | เดินไปมาตามเส้นทางสุ่ม |
| 🔴 ไล่ล่า | ตรวจพบผู้เล่น → วิ่งไล่ด้วยความเร็วสูง |
| 🟡 กลับจุดเริ่ม | หลุดจากระยะตรวจจับ → กลับไปลาดตระเวน |
| ⚪ หยุดนิ่ง | ยืนเฉยๆ แต่จะไล่ทันทีถ้าเข้าใกล้ |

### วิธีหลีกเลี่ยง:
- 🔦 **แสงไฟฉาย** อาจทำให้ศัตรูชะงัก (เมื่อส่องถูก)
- 👂 **ฟังเสียง** — เสียงบรรยากาศผีจะดังขึ้นเมื่อศัตรูอยู่ใกล้
- 🏃 **วิ่งหนี** — ออกจากระยะตรวจจับ 1.5 เท่า ศัตรูจะยอมแพ้
- ⚠️ **ถ้าถูกจับ → Game Over!**

---

## 🔦 ไฟฉาย

ตัวละครจะถือไฟฉายตลอดเวลา:

- ส่องเป็น **รูปกรวย** ไปทางที่คุณเดิน
- ช่วยให้มองเห็นทางในความมืด
- **บ้านที่มีไฟ** จะสว่างรอบๆ บ้าน

---

## 🗺️ แผนที่

- แผนที่แต่ละคืนจะ **แตกต่างกัน** ทั้งในเรื่องขนาดและการวางสิ่งกีดขวาง
- **สิ่งกีดขวาง** (ต้นไม้, กำแพง) ไม่สามารถเดินผ่านได้
- ใช้ **Minimap** ที่มุมจอเพื่อวางแผนเส้นทาง
- **ลูกศรชี้ทิศทาง** + **ระยะทาง** จะช่วยนำทางไปบ้านถัดไป

---

## ⏰ เวลา

- แต่ละคืนมี **จำกัดเวลา 6 ชั่วโมง** (ในเกม)
- เวลาในเกมเดินช้ากว่าเวลาจริง (สเกล 0.5x)
- เมื่อเหลือเวลา **1 ชั่วโมงสุดท้าย** จะมีข้อความเตือน ⚠️
- **หมดเวลา = Game Over!**

---

## 🎯 เคล็ดลับการเอาตัวรอด

1. **วางแผนเส้นทาง** — ดู Minimap ก่อนออกเดิน เลือกเส้นทางที่สั้นที่สุด
2. **จัดการพลังงาน** — อย่าวิ่งตลอด เดินสลับวิ่ง เพื่อเก็บพลังงานไว้หนีผี
3. **ฟังเสียงรอบข้าง** — เสียงบรรยากาศจะเตือนเมื่อผีอยู่ใกล้
4. **อย่ายืนนิ่ง** — ส่งพัสดุให้เร็วแล้วไปบ้านถัดไป
5. **จำเส้นทาง** — แผนที่ไม่เปลี่ยน ยิ่งเล่นบ่อยยิ่งจำได้
6. **ใช้ไฟฉาย** — ส่องไปทางที่ผีอยู่อาจทำให้มันชะงัก

## 🔧 ระบบเสียง

- 🎵 **เพลงประกอบ** — เล่นตลอดในทุกฉาก
- 🌲 **เสียงบรรยากาศป่า** — เสียงสุ่มเช่น กิ่งไม้หัก ลมพัด
- 👣 **เสียงเท้า** — เปลี่ยนตามเดิน/วิ่ง
- 🚪 **เสียงเคาะประตู** — เมื่อกด F ที่บ้าน
- 👻 **เสียงบรรยากาศผี** — ดังเมื่อศัตรูเข้าใกล้

## ❓ คำถามที่พบบ่อย

**Q: ทำไมวิ่งไม่ได้?**
A: พลังงานหมด! รอให้แถบพลังงานฟื้นตัวก่อน (หยุดนิ่ง = ฟื้นเร็ว)

**Q: หาบ้านไม่เจอ?**
A: ดูลูกศรชี้ทิศที่ด้านบนหน้าจอ และ Minimap ที่มุมขวาล่าง

**Q: คืนที่ 2/3 ปลดล็อคไม่ได้?**
A: ต้องเล่นคืนก่อนหน้าให้ผ่าน (ส่งพัสดุครบทุกบ้าน)

**Q: จะรีเซ็ตเซฟเกมยังไง?**
A: ที่หน้า Main Menu → กดปุ่ม "⚠️ รีเซ็ตความคืบหน้า"

**Q: เกมค้างหลัง Game Over?**
A: กดปุ่ม "คลิกเพื่อเล่นใหม่" หรือ "กลับไปเมนูหลัก" บนหน้าจอ Game Over
